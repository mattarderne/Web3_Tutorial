# Web3 Tutorial
## A hack towards building a smart-contract, for a hackathon.

> Tutorial following [this](https://coursetro.com/posts/code/99/Interacting-with-a-Smart-Contract-through-Web3.js-%28Tutorial%29). This Tutorial runs a local Ethereum TestRPC and allows you to interact with it through the browser.




To run
git clone %this_repo%
cd web3_test
npm init
npm install ethereum/web3.js --save
open file index.html in browser
